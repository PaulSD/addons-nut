# Home Assistant NUT-Latest Add-on Repository

This repo is not currently maintained.  The official NUT Add-on now includes the latest version of NUT, making this repo unnecessary.
However, this repo is being preserved for reference, since it may be useful as an example for extending other HA Add-ons.

This is a Home Assistant Add-on Repository which provides the [NUT-Latest](nut-latest) Add-on.

See the [NUT-Latest add-on documentation](nut-latest/DOCS.md) for more information.

To fork this repository and create your own copy, see [FORK.md](FORK.md).
